# Space tourism website (Frontend Mentor)

Meu primeiro website front-end, um multi-page! Este é um desafio que fiz do Frontend Mentor, posso dizer que tive dificuldades pra realiza-lo, contei com ajuda do IA várias vezes, porém sempre quebrando a cabeça muito antes de pedir ajuda, isso me fez aprender muito neste projeto.
O desafio em si era construir um multi-page, com um menu bem dinâmico, que muda de local, dependendo da página acessada, e também na responsividade como tablet / mobile, trazendo o menu-hamburger. O que me deixou mais contente em realizar foi a leitura de dados do json para as informações das páginas, dessa forma tive de construir uma renderização diferente para cada página, foi desafiador. Teve outros pontos interessantes como a alteração da posição das imagens via javascript, dependendo da largura da tela.

## Sobre
Foi um exercício muito difícil para mim, levei dias, e tive muita ajuda do IA, porém isso foi também benéfico pois eu aprendi muito. Eu realizei com meus conhecimentos até o momento adquirido no curso, a construção deste website, que me deixou muito feliz pois teve momentos que achei q teria de estudar mais para só então continuar com o desafio.
